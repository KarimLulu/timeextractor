Time Extractor








About

Time extractor API identifies and extracts time expressions such as time, date, time intervals, date intervals, durations, etc. from texts and html pages.
Below you can find API documentation

Chrome Extension

Chrome Extension was developed to demonstrate API possibilities.
Extension allows to identify on any web page dates/times and makes it possible to add them to google calendar.

Download Chrome Extension Chrome Extension User Guide

API Documentation:


Endpoints:

http://timeext.codeminders.com/timeextractor-2/api/annotate

https://timeext.codeminders.com:8443/timeextractor-2/api/annotate

You must POST the following parameters:


Parameter Description
Method POST
Data Params
id unique text identifier (integer)
email** Email which you entered during registration
key** API key that you received
timezone** Timezone offset
date** Current date in format, ex: 2014-10-24T13:37:40.931Z
text* Text to be annotated
html Html page to be annotated
rules_to_ignore UUID of rules that will be excluded,
ex. af400fa9-7306-4e87-8e75-9591ae532555;
Multiple ids should be separated with comma
Sample json params text 
[ {"id":"2",'text':" monday ", 'timezone_offset':"0", 
'date':'2014-10-24T13:37:40.931Z' },
 
{"id":"3",'text':" tuesday ", 'timezone_offset':"0",  
"rules_to_ignore":"364d8630-4592-4a24-9397-2634b0623bd74", 
'date':'2014-10-24T13:37:40.931Z' }];
Sample json params html*** 
[ {'html' : "<body _temporal_id = \"32\">24/02/2014</body>",  
'timezone_offset':"0", 'date':'2014-10-24T13:37:40.931Z' } ];

* either 'text' or 'html' parameter should be specified, otherwise 400 Exception Missing parameter is thrown
** obligatory parameter
*** for html page, each tag should have "_temporal_id" parameter, as it comes in response to identify tag with temporal text inside

API Response for Text


Response comes as object:
key - uniqueId
value - array of JsonObjects

Json Object’s fields
from Start position of annotation
to End position of annotation
locale Locale of annotated temporal expression
confidence Parameter that shows level of confidence of appearance exact word in the sentence (range: 0 to 1)
temporalType Type of extracted temporal object, one of four:
1. TIME_DATE
2. DATE_INTERVAL_TIME_INTERVAL
3. DURATION
4. DURATION_INTERVAL
5. SET
extractedTemporal Array of extarcted temporal object, can be one of 4 temporalTypes
look further under 'Extracted temporal object' table

API Response for Html


Response comes as object:
key - raw count of tags
value - array of JsonObjects

Json Object’s fields
from Start position of annotation
to End position of annotation
htmlFrom Start tag position in the html document
htmlTo End tag position in the html document
locale Locale of annotated temporal expression
confidence Parameter that shows level of confidence of appearance exact word in the sentence (range: 0 to 1)
tag Html tag for further search of result in html document
temporalType Type of extracted temporal object, one of four:
1. TIME_DATE
2. DATE_INTERVAL_TIME_INTERVAL
3. DURATION
4. DURATION_INTERVAL
5. SET
extractedTemporal Array of extarcted temporal object, can be one of 4 temporalTypes
look further under 'Extracted temporal object' table

JSON object's fields

id Rule id
example Example of text rule detects
name Rule name
confidence Parameter that shows level of confidence of appearance exact word in the sentence (range: 0 to 1)
