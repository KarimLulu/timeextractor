Temporal Extractor








About

Temporal extractor API identifies and extracts temporal expressions such as time, date, time intervals, date intervals, durations, etc. from texts and html pages. Below you can find API documentation:

API Request


URL http://ec2-54-81-15-231.compute-1.amazonaws.com:8080/timeextractor-0.0.2/api/annotate
Method POST
Data Params
id unique text identifier (integer)
text* Text to be annotated
html Html page to be annotated
Sample json params text 
var json = [
                {"id":"2",
                "text":"friday morning, saturday evening"
                },
                {"id":"3",
                "text":"saturday"
                }];
Sample json params html 
var json = [ {	'html' : "<body>2014-07-27 Monday <div>test saturday</div></body>"} ];
Sample call 
$.ajax({
		type : "POST",
		url : "http://ec2-54-81-15-231.compute-1.amazonaws.com:8080/timeextractor-0.0.2/api/annotate",
		data : JSON.stringify(json),
		crossDomain : true,
		contentType : "application/json",
		dataType :  "json",
                success : function(result) {
                             console.log(result);
                          }
        });
* either 'text' or 'html' parameter should be specified, otherwise 404 Exception is thrown
